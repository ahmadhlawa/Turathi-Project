# المجلد العام (Public)

هذا المجلد مخصص للملفات الثابتة (Static Assets) مثل الصور، الأيقونات، والخطوط.

- أي صورة تضعها هنا ستكون متاحة مباشرة عبر الرابط الجذري للموقع `/`.
- مثال: إذا وضعت صورة باسم `palestine.jpg` هنا، يمكنك استخدامها في الكود بهذا الشكل:
`<img src="/palestine.jpg" alt="فلسطين" />`
